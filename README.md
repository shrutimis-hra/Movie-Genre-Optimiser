# Movie-Genre-Optimiser


A machine learning project designed to automatically predict movie genres based on textual metadata or visual elements (like movie posters). 

# 📌 Project Overview
Manually tagging movie genres can be time-consuming and subjective. This project builds an end-to-end pipeline to analyzeand classify movies into their respective genres using Natural Language Processing (NLP) 

This is a multi-label classification problem, as movies frequently belong to multiple genres (e.g., Sci-Fi + Action + Thriller).
